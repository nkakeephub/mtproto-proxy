**Настраиваем MTProto прокси с Fake TLS**


Проверка обновлений && установка обновлений
```
sudo apt update && sudo apt upgrade -y
```

Ставим MTproxy
```
bash <(curl -s "https://raw.githubusercontent.com/nkakeephub/mtproto-proxy/main/start-mtproxy.sh")
```

Проверяем работает ли MTproxy

    sudo docker ps

В ответе мы должны увидеть STATUS UP
